# AgentGate Development Notes

AgentGate is a focused local TypeScript CLI. Keep the product centered on tool-call firewalling, trace recording, and replay-based regression tests.

Do not add a SaaS dashboard, database, telemetry, hosted service, or agent framework.

Use pnpm, ESM, Vitest, tsup, ESLint, and Prettier. Keep implementation choices deterministic and easy to test.
